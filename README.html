📖 About

Luminaries of Islam is a fully offline, privacy-first, interactive web app that teaches children and young learners about 12 groundbreaking Muslim scientists from the Islamic Golden Age (8th–14th century CE).

  • Islamic geometric design — non-figurative, calligraphy-inspired UI
  • Trilingual — Arabic (with full RTL), English, French
  • Three knowledge levels — Newb, Explorer, Developer
  • Voice interaction — Speech-to-Text search + Text-to-Speech read-aloud
  • Gamified quizzes — badges, instant feedback, reflection prompts
  • Zero dependencies — pure HTML/CSS/JS, no build step, no server

🚀 Installation — Step by Step

Option A: Direct Download (Simplest)

  1. Download the ZIP file: luminaries-of-islam-v1.3.zip
  2. Extract the ZIP to any folder on your computer
  3. Open index.html in your web browser (double-click it)
  4. Done! No server, no terminal, nothing to install

Option B: Clone from Git

# Step 1: Clone the repository
git clone https://github.com/YOUR-USERNAME/luminaries-of-islam.git

# Step 2: Enter the project folder
cd luminaries-of-islam

# Step 3: Open in your browser
open index.html          # macOS
xdg-open index.html      # Linux
start index.html         # Windows

Option C: Local Server (Optional)

# Python 3:
python3 -m http.server 8000
# Open → http://localhost:8000

# Or Node.js:
npx serve .
# Open → URL shown in terminal

🎯 Features

🌍 Trilingual Interface

Language switcher in the nav bar: عر / EN / FR. All content switches instantly — UI labels, scientist bios, quiz questions, feedback, badges. Arabic activates full RTL layout. Preference saved in localStorage.

🎓 Three Knowledge Levels

LevelIconStyle
Newb🟢Fun analogies, simple language
Explorer🔵Technical vocabulary, historical context
Developer🟣Deep analysis, precise data, academic detail

🔊 Text-to-Speech (TTS)

  • 🔊 button on cards, modals, timeline, quiz, welcome
  • Floating control bar: Play/Pause, Stop, Speed (🐢 0.7x / 🚶 1x / 🐇 1.4x)
  • Browser-native SpeechSynthesis API — no external data transfer

🎙️ Speech-to-Text (STT)

  • 🎤 on home search bar — say a scientist's name
  • 🎤 in quiz — say your answer aloud
  • Browser-native SpeechRecognition API — no voice data stored
  • Graceful fallback: mic buttons hidden if browser doesn't support it

🏆 Quiz / Challenge Zone

  • 12 questions, 8 randomly selected per session
  • Trilingual questions, options, and feedback
  • Badges: 🌱 Young Scholar → 📚 Knowledge Seeker → 🌟 Golden Age Expert
  • End-of-quiz reflection prompts

📅 Interactive Timeline

Horizontal scroll from 8th to 14th century. Click any scientist dot to open their profile. TTS on each point.

🛠️ How-To Guide

How to Use the App

  1. Open index.html in your browser
  2. Choose language: عر (Arabic), EN (English), or FR (French)
  3. Choose level: 🟢 Newb, 🔵 Explorer, or 🟣 Developer
  4. Browse scientists — filter by field or scroll the grid
  5. Click any card to read the full profile
  6. Try voice search: click 🎤 and say a scientist's name
  7. Click 🔊 to hear content read aloud
  8. Explore the Timeline to see scientists across centuries
  9. Take the Quiz in the Challenge Zone
  10. Earn the 🌟 Golden Age Expert badge!

How to Add a New Scientist

  1. Open js/app.js — add to the scientists array
  2. Open js/i18n.js — add translations to scientistI18n
  3. Add quiz questions to quizI18n array in js/i18n.js
  4. Add filter button in index.html if new field category
  5. Increment version, update CHANGES.md

How to Add a New Language

  1. Open js/i18n.js
  2. Add a new key to every translation object (e.g., es for Spanish)
  3. Add a language button to the switcher in index.html
  4. Add scientist + quiz translations for the new language
  5. For RTL languages, add the code to the RTL check in i18n.js

How to Change Colors

Edit CSS variables at the top of css/style.css:

:root {
    --teal: #0A6E6E;      /* Primary */
    --gold: #C9A84C;       /* Accent */
    --cream: #FDF6EC;      /* Background */
    --charcoal: #2C2C2C;   /* Text */
}

🏗️ Architecture

┌─────────────────────────────────────────────┐ │ index.html │ │ (structure, layout, semantic HTML) │ ├─────────────────────────────────────────────┤ │ css/style.css │ │ (design, animations, RTL, responsive) │ ├─────────────┬──────────┬──────────┬─────────┤ │ js/i18n.js │ js/app.js│ js/tts.js│js/stt.js│ │ (language) │ (logic) │ (speech) │ (voice) │ └─────────────┴──────────┴──────────┴─────────┘
Key design decisions: No framework (max simplicity), separate i18n module (clean content/logic split), browser Speech APIs (privacy-first), CSS-only patterns (no image assets), localStorage only (no cookies, no server), emoji icons (universal, no assets).
